About Us

Who We Are

The Skill Development Center for Experiential Learning and Global Engagement (SKILL E.L.G.E) is a dynamic student-driven initiative at NMIMS Hyderabad Campus dedicated to bridging the gap between academic knowledge and practical industry experience.

We believe in learning by doing. Through workshops, hands-on projects, industry collaborations, and global exposure programs, we empower students to develop critical skills that go beyond the classroom.

Innovation Focus

Encouraging creative problem-solving and entrepreneurial thinking

Global Perspective

Connecting students with international opportunities and networks

Industry Connect

Building bridges between academia and industry leaders

Our Mission

The mission of SKILL E.L.G.E is to foster holistic student development by promoting hands-on learning, industry exposure, and global perspectives that prepare students for careers, leadership, and lifelong learning.

Our Vision

To be a leading student-driven skill development platform that transforms learners into globally competent, industry-ready, and socially responsible professionals through experiential learning and global engagement.

Python Full Stack Development Workshop

Workshop Schedule Overview

Day 1 (10:00 AM – 5:00 PM)

  • Full Stack & Industry Overview
  • Modern JavaScript (ES6+)
  • React Fundamentals & Hooks
  • Routing & REST API Consumption

Day 2 (10:00 AM – 5:00 PM)

  • Python Backend with Django & REST APIs
  • Database Integration
  • Full Stack Integration & Security (JWT)
  • MNC Technologies: APIs, WebSockets, Deployment, Architecture

Software Requirements

Must Be Installed Before Workshop:

  • Python 3.10+
  • Visual Studio Code
  • Node.js (LTS)
  • Git
  • Django
  • Django REST Framework
  • Postman
  • Google Chrome Browser

Student Readiness Checklist

  • Laptop with minimum 8 GB RAM
  • Stable internet connection
  • All software installed and verified
  • Basic understanding of Python & JavaScript
  • GitHub account created (optional but recommended)
  • Workshop folder created (frontend / backend)
Important: Students are advised to complete installations before Day 1 to ensure full participation in hands-on sessions.
